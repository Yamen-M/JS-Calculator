# JS-Calculator

This is a fully functional calculator built as the final project of [The Odin Project](https://www.theodinproject.com/) Foundations Curriculum. It performs basic arithmetic operations and supports both **keyboard** and **mouse** input.

## 🚀 Features

- Basic arithmetic: `+`, `-`, `*`, `/`
- Clear all (`AC`) and backspace (`⌫`)
- Keyboard and mouse input supported
- Dynamic display resizing for large numbers
- Responsive and clean UI
- Built with custom DOM manipulation

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
